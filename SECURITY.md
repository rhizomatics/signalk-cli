# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Use GitHub issues to report security problems, unless you believe that disclosure publically is risky, in which
case email cyber@rhizomatics.org.uk.
